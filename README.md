🛡️ Monitor de Risco Operacional
Projeto desenvolvido para simular um motor de regras de risco financeiro.

🚀 Tecnologias
Java 17/21

Spring Boot 3

Spring Data JPA (H2 Database)

Lombok

⚙️ Regras de Risco Implementadas
Filtro de Valor: Transações acima de R$ 10.000,00 são marcadas como SUSPEITA.

Validação Automática: Uso de arquitetura em camadas para garantir a integridade dos dados.
